Jenis Bahasa Pemrograman Web

Istilah pemrograman web terdiri atas dua kata, yaitu pemrograman dan web. Berikut ini adalah definisi dari kedua kata tersebut :

Pemrograman
  Proses, cara, perbuatan program

Web
  Jaringan komputer yang terdiri dari kumpulan situs internet yang menawarkan teks dan grafik dan suara dan sumber daya animasi melalui protokol transfer hypertext
  1. Client Side Scripting
    Contoh:
    • HTML
    • CSS
    • Javascript
  2. Server Side Scripting
    Contoh:
    • PHP
    • ASP
    • Java Server Page
